# BodyLog

Личный трекер: вес, дозировки (инъекции), питание, активность и аналитика с графиками.

## Стек

- Next.js 14 (App Router), TypeScript
- TailwindCSS, shadcn/ui
- Recharts, React Hook Form + Zod
- Dexie (IndexedDB), date-fns
- PWA (next-pwa)

## Запуск

```bash
npm install
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000). Данные хранятся локально в браузере (offline-first). Экспорт/импорт — в Настройках.

## Функции

- **Дашборд**: текущий вес, Δ за 7/30 дней, серия ввода, подсказка следующей инъекции, график веса за 30 дней со скользящим средним.
- **Лог**: таблица по дням (вес, ккал, макросы, дозы, активность), фильтры.
- **Аналитика**: вкладки — вес, питание (калории + макросы), дозы, активность.
- **Настройки**: профиль и цели, светлая/тёмная тема, экспорт JSON/CSV, импорт JSON с превью.

Все формы поддерживают быстрый ввод (Сегодня/Вчера), валидацию и отмену после добавления (toast).
